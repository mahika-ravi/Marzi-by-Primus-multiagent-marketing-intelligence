# Multi-Agent Marketing Intelligence System
### Marzi by Primus | AI Intern Project

> 🚧 Active development — work in progress.

## Overview
A 7-agent agentic AI system for end-to-end digital marketing automation, 
built for Marzi by Primus (backed by General Catalyst & Nikhil Kamath).

## Agent Architecture
| Agent | Role |
|---|---|
| Competitive Intel Agent | Monitors brand + competitor social signals |
| Content Strategy Agent | Surfaces high-TAM content ideas daily |
| Hook & Script Agent | Generates platform-specific hooks and scripts |
| Trend Agent | Identifies real-time trending topics in niche |
| Scheduling Agent | Plans content calendar based on performance data |
| Distribution Agent | Automates delivery via MS Teams workflows |
| Performance Agent | Tracks campaign metrics via CleverTap + LinkRunner |

## Tech Stack
- **Orchestration:** Python, LangChain / LlamaIndex
- **LLMs:** Claude (Anthropic)
- **Automation:** MS Teams Workflows
- **MarTech:** CleverTap, LinkRunner
- **Delivery:** Daily automated briefs to company GC
